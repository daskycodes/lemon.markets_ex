    LemonMarketsEx (LemonMarketsEx v0.1.0) View Source

    Elixir client for the lemon.markets API.

    This module provides all requests to the lemon.markets API.

    Please check the README for more information and refer to the Modules LemonMarketsEx.Authentication, LemonMarketsEx.Data, LemonMarketsEx.Orders, LemonMarketsEx.Spaces and LemonMarketsEx.TradingVenues for more details and examples.
