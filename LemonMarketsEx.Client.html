    LemonMarketsEx.Client (LemonMarketsEx v0.1.0) View Source

    Client for interacting with the lemon.markets API.

    After sucessfully authenticating with the API, and receiving the LemonMarketsEx.Token.t/0 from LemonMarketsEx.Authentication.request_auth_token/2, you can call LemonMarketsEx.Client.new/2 to get a client, scoped to a space to make further API calls.

    For more information about the Authentication process, see here.
